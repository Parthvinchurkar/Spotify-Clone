# 🎵 Spotify Clone

A simple Spotify Clone built using React, Redux Toolkit, and Vite.

## 🚀 Features
- Play / Pause Songs
- Next / Previous
- Shuffle & Repeat
- Like Songs
- Volume Control
- Dynamic Progress Bar

## 🛠 Tech Stack
React, Redux Toolkit, Vite, Uno css

## 📦 How to Run This Project

1. Clone the repository:
git clone https://github.com/Parthvinchurkar/spotify-clone.git

2. Go to project folder:
cd spotify-clone

3. Install dependencies:
npm install

4. Start development server:
npm run dev

5. Open in browser:
http://localhost:5173
